Neben offensichtlichen Problemen wie der Stummschaltung des Sound Ihres Computers ist eine moegliche Ursache, dass Ihr Webbrowser dies nicht unterstuetzt. Es gibt zwei voellig unterschiedliche Techniken, um den Sound abzuspielen. Ihr Computer muss (mindestens) einen von ihnen unterstuetzen, damit der Sound funktioniert: Heutzutage am haeufigsten verwendet: HTML5 WebAudio, da dies in den meisten Webbrowsern seit mehreren Jahren verfuegbar ist: Firefox Version 17 oder hoeher Chrome Version 10 oder hoeher Safari Version 6 oder hoeher Opera Version 15 oder hoeher Edge (wie im Lieferumfang von Windows 10 enthalten) Leider unterstuetzt Internet Explorer diese Technologie noch nicht. Im Wesentlichen nur fuer Internet Explorer koennen Sie auch Java-Applets verwenden. Dazu benoetigen Sie nicht nur einen Webbrowser, sondern auch Version 1.4.2 oder eine neuere Version von Java, die auf Ihrem Computer ausgefuehrt wird. Versuchen Sie http://javatester.org/version.html herauszufinden, ob Sie Java haben und um welche Version es sich handelt. Moeglicherweise muessen Sie Java-Applets pro Site aktivieren. siehe hier.
